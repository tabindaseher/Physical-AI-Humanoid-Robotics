# Appendix B: Gazebo Setup

## Installing Gazebo

### System Requirements
- Ubuntu 20.04 or later (or equivalent Linux distribution)
- At least 8GB RAM recommended
- GPU with OpenGL 3.3 support

### Installation Steps
1. Add the Gazebo repository to your system
2. Install the required packages
3. Verify the installation

### Basic Configuration
- Setting up environment variables
- Configuring graphics drivers
- Performance optimization settings

## Creating Your First Simulation

### Basic Robot Model
- URDF (Unified Robot Description Format) files
- Link and joint definitions
- Physical properties

### World Setup
- Creating environments
- Adding objects and obstacles
- Lighting and camera configuration
