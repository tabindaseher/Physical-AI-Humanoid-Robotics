# Appendix E: Development Resources

## Software Resources

### Essential Tools
- ROS 2 (Humble Hawksbill or later)
- Gazebo/IGNITION for simulation
- NVIDIA Isaac ROS packages
- Git for version control
- VS Code or similar IDE with ROS extensions

### Libraries and Frameworks
- OpenCV for computer vision
- TensorFlow/PyTorch for AI
- MoveIt for motion planning
- PCL (Point Cloud Library) for 3D processing

## Hardware Resources

### Recommended Platforms
- NVIDIA Jetson for edge AI
- Robot hardware platforms (TurtleBot3, Fetch, etc.)
- Sensor packages (cameras, LiDAR, IMU)
- Actuators and motor controllers

## Online Resources

### Documentation
- ROS 2 documentation
- Gazebo tutorials
- Isaac ROS guides
- MoveIt tutorials

### Communities
- ROS Discourse
- NVIDIA Developer Forums
- GitHub repositories
- Stack Overflow
