# Appendix C: Isaac ROS Tutorials

## Getting Started with Isaac ROS

Isaac ROS is a collection of hardware accelerated software packages that bring NVIDIA GPU acceleration to ROS 2. This appendix provides tutorials for common Isaac ROS packages.

### Prerequisites
- NVIDIA GPU with CUDA support
- CUDA toolkit installed
- ROS 2 environment set up
- Isaac ROS packages installed
