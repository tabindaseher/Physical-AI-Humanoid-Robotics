# Contributing to Physical AI & Humanoid Robotics

Thank you for your interest in contributing to the Physical AI & Humanoid Robotics project! All contributions are welcome.

## üõ†Ô∏è How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issues to report bugs
- Fill out the bug report template with:
  - Clear title and description
  - Steps to reproduce
  - Expected vs actual behavior
  - Screenshots if applicable
  - Environment details (OS, browser, node/python versions)

### 2. Suggesting Features
- Open an issue with the feature request template
- Explain the problem you're trying to solve
- Describe your proposed solution
- Include any relevant examples or references

### 3. Code Contributions
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Add tests if applicable (especially for backend changes)
5. Ensure your code follows the project's style
6. Commit your changes with clear, descriptive messages
7. Push to your fork (`git push origin feature/amazing-feature`)
8. Open a pull request

## üìù Code Guidelines

### Backend (Python)
- Follow PEP 8 code style
- Use type hints for all function parameters and return values
- Write docstrings for all public functions
- Add error handling where appropriate

### Frontend (JavaScript/React)
- Use functional components with hooks
- Follow React best practices
- Keep components small and focused
- Use TypeScript for new components

### Documentation (Markdown)
- Use clear, concise language
- Follow the existing documentation style
- Include examples where helpful

## üöÄ Pull Request Process

1. **Before creating a PR:**
   - Update the README.md if needed
   - Add tests for new functionality
   - Ensure all tests pass
   - Update documentation if applicable

2. **When creating a PR:**
   - Use a clear title describing the changes
   - Include a detailed description
   - Link any relevant issues
   - Add appropriate labels

3. **After submission:**
   - Be responsive to feedback
   - Make requested changes
   - Update the PR as needed
