# AI-Native Textbook: Physical AI & Humanoid Robotics

This is an AI-native, interactive technical textbook designed to teach a full university-level course on Physical AI & Humanoid Robotics. The book bridges artificial intelligence, robotics middleware, simulation, embodied cognition, and vision-language-action systems.

## Overview

This textbook is designed for:
- Senior undergraduate and graduate students
- AI engineers transitioning into robotics and embodied intelligence
- Robotics engineers integrating LLMs and cognitive planning
- Educators and curriculum designers

## Structure

The textbook is organized into the following modules:
1. Foundations of Physical AI & Embodied Intelligence
2. ROS 2 as the Robotic Nervous System
3. Digital Twins with Gazebo & Unity
4. AI-Robot Brain with NVIDIA Isaac
5. Vision-Language-Action Pipelines
6. Conversational & Multimodal Robotics
7. Capstone: Autonomous Humanoid System Architecture

## Features

- Interactive RAG (Retrieval-Augmented Generation) chatbot for answering questions
- Academic rigor with peer-reviewed citations
- Simulation-first approach using Gazebo, Unity, and NVIDIA Isaac Sim
- Content optimized for both human learning and AI retrieval

## Technology Stack

- **Frontend**: Docusaurus v3+ (React-based static site generator)
- **AI/ML**: OpenAI API, Embedding models for RAG
- **Backend**: FastAPI (Python web framework)
- **Database**: Neon Serverless Postgres
- **Vector DB**: Qdrant Cloud
- **Deployment**: GitHub Pages

## Getting Started

To run the textbook locally:

1. Clone the repository
2. Navigate to the `textbook` directory
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

The textbook will be available at http://localhost:3000
